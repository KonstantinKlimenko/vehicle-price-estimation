# Vehicle Price Estimation

Проект по машинному обучению для оценки рыночной стоимости автомобилей с пробегом по техническим характеристикам и комплектации.

## Задача

Нужно было построить модель регрессии и сравнить несколько алгоритмов по трем критериям:

- качество предсказания;
- время обучения;
- скорость инференса.

Целевая метрика проекта: `RMSE < 2500`.

## Что сделано

- выполнена очистка данных и обработка пропусков;
- удалены объекты с нулевой ценой и явными аномалиями;
- обучены и сравнены модели:
  - `LinearRegression`
  - `RandomForest`
  - `LightGBM`
  - `CatBoost`
  - `XGBoost`
- выполнен ручной подбор параметров для `LightGBM` и `CatBoost`;
- проведено финальное тестирование лучшей модели.

## Результаты

Лучшая модель: **LightGBM**

### Метрики на валидации

| Model | RMSE | Train time, sec | Predict time, sec |
|---|---:|---:|---:|
| LightGBM | 1521.16 | 3.42 | 0.41 |
| XGBoost | 1569.13 | 1.74 | 0.18 |
| CatBoost | 1624.66 | 31.99 | 0.12 |
| RandomForest | 1694.29 | 35.31 | 0.11 |
| LinearRegression | 2568.17 | 2.00 | 0.06 |

### Метрики финальной модели на тесте

- RMSE: **1499.14**
- MAE: **933.83**
- Median AE: **561.84**
- Время обучения: **3.97 сек**
- Время предсказания: **0.424 сек**

Требование задачи по качеству выполнено с большим запасом.

## Наиболее важные признаки

По итогам обучения финальной модели наибольший вклад в предсказание стоимости внесли:

1. `Power`
2. `RegistrationYear`
3. `Kilometer`
4. `Model`
5. `Repaired`
